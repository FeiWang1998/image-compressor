# 图片压缩工具 (Image Compressor)

一个简单易用的在线图片压缩工具，无需安装任何软件，直接在浏览器中使用。

## 功能特点

- 🖼️ 支持拖拽或点击上传图片
- 👀 实时预览原图和压缩后的图片
- 🎚️ 可调节压缩质量（0-100%）
- 📊 显示图片详细信息（尺寸、大小）
- ⬇️ 支持下载压缩后的图片
- 💡 简洁直观的用户界面

## 技术栈
- HTML5
- CSS3
- JavaScript (原生)

## 使用方法
1. 直接访问网页
2. 拖拽图片或点击上传按钮选择图片
3. 调节压缩质量滑块
4. 点击下载按钮保存压缩后的图片

## 浏览器支持
- Chrome (推荐)
- Firefox
- Safari
- Edge
